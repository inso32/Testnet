
## **Useful commands**

List all keys

```
gaiad keys list
```

Find out the address
~~~
gaiad keys show Atlas.Node -a
~~~

How to find out address and voloperaddress using explorer?

Open the validator and click on the transaction at the bottom in the Transactions section.
